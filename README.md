# 端到端的大模型安全推理系统

Java Swing 演示版。界面按修改说明文档调整为数据拥有者和数据消费者两个角色，采用左侧用户信息、右侧分组功能区的报告式界面。

## 功能

- 数据拥有者登录与导航主页。
- 生产者管理：维护输入来源信息和输入提交信息。
- 安全加密：对待处理输入进行本地安全处理，输出密文输入。
- 密文上传：把密文输入提交至密文存储服务器，并展示上传状态。
- 存储内容：查看已存储的密文输入记录。
- 数据消费者登录后进入推理结果查询界面。
- 推理结果查询：创建查询任务，查看任务状态、结果编号、处理时间和结果内容。

## 运行

```powershell
javac -encoding UTF-8 -d out (Get-ChildItem -Recurse -Filter *.java -Path src | ForEach-Object { $_.FullName })
java -cp out privacyguard.Main
```
